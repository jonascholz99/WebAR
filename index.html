WebAR Examples
This webpage is a comprehensive resource for exploring WebAR, featuring an array of tests that delve into the vast potential of Augmented Reality on the web. Each example on the page uniquely incorporates different technologies, showcasing the versatility and wide-ranging applicability of WebAR.

Source code hosted at GitHub.

ARjs & A-Frame - Image Tracking

This example is an implementation of image tracking and the display of a GLTF model with AR.js and A-Frame (Marker: Image).


ARjs & ThreeJS - Marker Tracking

In this example, marker tracking is used with AR.js and Three.js. A simple cube is displayed on the marker. (Marker: Hiro)


ARjs & ThreeJS - 3D Model

Here, a 3D object is also displayed, but not through image tracking. Instead, marker tracking with AR.js and Three.js is used. (Marker: Kanji)
